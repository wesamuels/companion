# Companion to the Statistical Curriculum for the Nursing PhD and DNP

This "book" is a companion to the statistics courses offered as part of the [Ph.D. in Nursing Research program](http://www.hunter.cuny.edu/nursing/admissions/graduate/phd-in-nursing) at [Hunter College](https://hunter.cuny.edu/), [CUNY](https://www.cuny.edu/). It also provides supplemental materials for Hunter's [Doctor of Nursing Practice program](https://www.hunter.cuny.edu/nursing/admissions/graduate/dnp).
